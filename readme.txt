Send the custom email to the logged in users.

== Description ==

This plugin include cron timer of one minute. Send the emails in batch vise to the logged in users. Also provide wysiwyg editor for create custom email template.

== Features ==

1. Create your custom email template according to your requirements.
2. Provide email testing section.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/email-scheduling` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
